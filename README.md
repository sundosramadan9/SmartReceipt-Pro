An advanced bill management application built with pure JavaScript that simulates a real-world checkout system with smart discount rules.

🌟 Key Features
Dynamic Calculations: Automatically calculates tax and tips based on user input.

Smart Promo System: Includes a logic-based discount engine (VIP & STUDENT codes).

Split Bill Utility: A feature to divide the total amount among multiple people.

Modern UI: Fully responsive and clean English interface for a professional user experience.

🛠️ Technologies Used
HTML5 (Structure)

CSS3 (Professional Dashboard Styling)

JavaScript (ES6+) (Mathematical Logic, DOM Manipulation, and Conditionals).

💡 Logic Implemented
This project demonstrates advanced use of JavaScript Logical Operators (&&, ||) and Arithmetic Operators to handle complex business rules for discounts and financial totals.
